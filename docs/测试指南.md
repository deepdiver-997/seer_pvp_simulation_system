# 赛尔号对战模拟器 - 测试指南

## 启动服务器

```bash
cd /path/to/Simulation-battle-system
./build/sim_server
```

服务器会在 `127.0.0.1:4399` 监听。

## 测试客户端

### Node.js 客户端

```bash
node docs/test-battle.js
```

**功能:**
- 连接服务器
- 发送心跳包
- 初始化对战 (波克尔x6 vs 波克尔x6)
- 自动发送心跳保持连接

**修改对战配置:**

编辑 `docs/test-battle.js` 中的 `initBattle()` 函数:

```javascript
const monsterId = 12;  // 精灵ID

```

## 协议说明

**消息格式 (二进制, Big-Endian):**

```
[4 bytes: total_length][2 bytes: command][4 bytes: uuid][params...]
```

**命令类型:**

| Command | 值 | 说明 |
|---------|-----|------|
| INVALID | 0 | 无效命令 |
| SELECT_SKILL | 1 | 选择技能 |
| USE_MEDICINE | 2 | 使用药品 |
| CHOOSE_PET | 3 | 切换精灵 |
| SEND_EMOJI | 4 | 发送表情 |
| INIT_BATTLE | 5 | 初始化对战 |
| SYNC_STATE | 10 | 同步状态 |
| HEARTBEAT | 11 | 心跳 |

**BattleCreateRequest 格式 (576 bytes):**

```
12个精灵 x (pet_id + 5个skill_id + 6个base属性) = 576 bytes
每个精灵: pet_id(4) + skill_id[5](20) + base[6](24) = 48 bytes
```

## 已实现的效果

### Effect 4 - 自身属性变化
- 20017 高速移动: 速度+2

### Effect 5 - 对手属性变化
- 20002 鸣叫: 对手攻击-1
- 20016 吹飞: 对手速度-1

### Effect 6 - 反弹伤害
- 10038 突进: 1/4伤害反弹
- 10039 猛禽: 1/4伤害反弹

### Effect 7 - 同生共死
- 10036 同生共死: 对方HP>自己时, 对方HP=自己HP

### Effect 8 - 手下留情
- 10057 手下留情: 伤害大于对方HP时, 对方剩余1HP

## 测试流程

1. 启动服务器: `./build/sim_server`
2. 运行客户端: `node docs/test-battle.js`
3. 观察战斗流程 (状态 5-37)
4. 结束: Ctrl+C

## 动态库

效果函数从以下动态库加载:
- `resources/moves_lib/liblib_1.dylib` (macOS) / `.so` (Linux) / `.dll` (Windows)
- `resources/soul_lib/liblib_1.dylib`

插件导出函数:
- `skill_plugin_register` - 注册技能效果
- `soulmark_plugin_register` - 注册魂印效果
