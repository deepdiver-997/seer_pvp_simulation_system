# 赛尔号PVP对战模拟系统

## 项目概述
本项目是一个基于C++实现的赛尔号PVP对战模拟系统，模拟赛尔号精灵对战的核心逻辑，包括精灵属性、技能效果、状态异常等机制。目前状态机逻辑尚未完全实现，主要是本赛蛆对于各种判定还不是很清楚，希望各位赛皇多多指教。

## 功能模块
1. **对战系统**：模拟精灵对战的核心逻辑，包括技能释放、属性克制、状态异常等。
2. **精灵管理**：支持精灵的属性、技能、状态管理。
3. **状态异常系统**：实现精灵的异常状态（如中毒、麻痹等）及其效果。
4. **效果模块**：定义了一些基础的、常被复用的效果，用于支持精灵的技能效果，现在还没实现。
5. **数值计算**：处理战斗中的伤害计算、属性加成等。
6. **服务器支持**：提供对战服务器的基本功能，支持多客户端连接。

## 技术栈
- **编程语言**：C++
- **网络库**：Boost.Asio
- **数据库**：MySQL（通过连接池管理）
- **线程池**：自定义线程池实现
- **状态机**：基于有限状态机（FSM）的战斗流程控制
